# Portuguese Players in the Premier League

Growing up in 1999/2000, I always loved the Premier League. As a kid looking up to English football, seeing Portuguese players in that league filled me with pride. It was something I followed very closely — mostly because I remember playing with my friends pretending I was Newcastle's Hugo Viana.

This project is a tribute to every Portuguese player who has stepped onto a Premier League pitch, season by season.
