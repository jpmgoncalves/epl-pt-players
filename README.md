# Portuguese Players in the Premier League

I always loved the Premier League. As someone who was growing up in 1999/2000 looking up to English football, seeing Portuguese players in that league filled me with pride. So this is something I always followed very closely, mostly because I remember as a kid playing with my friends as if I was Newcastle's Hugo Viana.

This project is a visual timeline of every Portuguese player who has appeared in the Premier League, from the 1995/96 season to the present day.

## Live Website

[epl-pt-players.vercel.app](https://epl-pt-players.vercel.app/)

## Data

All data is sourced from the CSV file included in this repository:

[Download Portuguese EPL Players by Season (.csv)](https://github.com/jpmgoncalves/epl-pt-players/blob/main/Portuguese%20EPL%20Players%20by%20Season.csv)

The dataset covers **30 seasons** (1995/96 — 2025/26) and **312 player–season entries** across multiple clubs.
